# Manual de Usuario: Conexión Segura a la VPN Corporativa

**Versión:** 1.0
**Fecha:** Noviembre 2023

## Introducción

Para mejorar la seguridad de nuestra red, hemos añadido un paso de verificación a la conexión de la VPN. A partir de ahora, necesitarás un código de seguridad de 6 dígitos, además de tu conexión habitual.

Este manual te guiará en dos sencillos pasos:

1.  **Configurar tu teléfono** para generar los códigos de seguridad.
2.  **Usar la nueva aplicación** para conectarte a la VPN.

---

## Paso 1: Configurar tu Aplicación de Autenticación

Este es un paso que **solo necesitarás hacer una vez**.

Para generar los códigos de seguridad, usarás una aplicación en tu teléfono móvil. Te recomendamos **Google Authenticator**, pero puedes usar cualquier otra que soporte códigos QR (como Authy, Microsoft Authenticator, etc.).

### ¿Cómo configurarlo?

1.  **Instala la Aplicación:**
    - Ve a la tienda de aplicaciones de tu teléfono (App Store para iPhone, Google Play para Android) y descarga `Google Authenticator` o tu aplicación preferida.

2.  **Escanea el Código QR:**
    - Tu administrador de TI te proporcionará un **código QR único**.
    - Abre la aplicación de autenticación en tu teléfono y busca la opción para añadir una nueva cuenta (normalmente un botón con un `+`).
    - Elige la opción "Escanear código QR" y apunta la cámara de tu teléfono al código que te dieron.

3.  **¡Listo!**
    - La aplicación reconocerá el código y añadirá una nueva entrada llamada "VPN Corporativa" (o similar).
    - Verás que muestra un **código de 6 dígitos que cambia cada 30 segundos**. Este es el código que usarás para conectarte.

---

## Paso 2: Conectarse a la VPN

Ahora que ya tienes tu teléfono listo, el proceso para conectarte a la VPN es muy sencillo.

1.  **Abre la Aplicación `VPNConnect`:**
    - Haz doble clic en el icono de la aplicación que te proporcionó el departamento de TI.

2.  **Introduce el Código de 6 Dígitos:**
    - Verás una nueva casilla llamada "**Código 2FA**".
    - Abre la aplicación de autenticación en tu teléfono y mira el código actual para la "VPN Corporativa".
    - Escribe ese código en la casilla.

    > **¡Importante!** El código cambia cada 30 segundos. Asegúrate de usar el que estás viendo en ese momento.

3.  **Haz Clic en "Conectar":**
    - Una vez introducido el código, presiona el botón "Conectar".
    - La aplicación verificará tu código, realizará la conexión segura y te notificará cuando estés conectado.

---

## Solución de Problemas Comunes

- **"Mi código no funciona" o "Código inválido"**:
    - **Revisa la hora de tu teléfono.** La causa más común de este problema es que la hora de tu teléfono no está sincronizada. Asegúrate de que la fecha y hora estén configuradas en "Automático" en los ajustes de tu teléfono.
    - **Espera a que el código cambie.** Si introdujiste un código justo cuando estaba a punto de expirar, es posible que no haya sido válido. Espera al siguiente código e inténtalo de nuevo.

- **"No puedo conectarme, pero mi código es correcto"**:
    - Asegúrate de tener una conexión a internet estable.
    - Si el problema continúa, por favor, contacta con el departamento de Soporte de TI.
